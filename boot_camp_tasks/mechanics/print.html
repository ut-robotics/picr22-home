3D print

. Install PrusaSlicer https://www.prusa3d.com/page/prusaslicer_424/
version , do not install any of the “Drivers (for updating firmware)”.

  1. In Fusion 360, select File > 3D Print. This will open up a prompt, select the model you want to export, for the format choose STL (Binary), and Unit Type as Millimeters. Click “OK” and a prompt should appear to save the .stl file. Save it somewhere that you can access easily.

  2. Open PrusaSlicer. A configuration wizard should appear. Click “Next”. Select these options when prompted:

    1. Prusa FFF > Original Prusa i3 MK3S & MK3S+ > 0.4 mm nozzle

    2. Filaments > Printer: (All) > Type: PLA > Vendor: Made for Prusa >

    3. Profile: Prusa PLA

    4. View mode > Advanced mode

    5. Click “Finish”

  3. Import the earlier saved .stl file from File > Import > Import STL/OBJ/… .

  4. The model might be imported in a not ideal orientation for printing. Click on the model, then use “Place on face” to select a face that will be set against the print bed. image::../images/prusa_place.png[]

  5. Under “Print settings” select “0.3mm DRAFT”.

  6. At the bottom right, select “Slice now”. The model will be “sliced” and show how the filament will be layed out.

  7. At the bottom right, select “Export G-code, which will prompt you to save the file. Save it somewhere that you can access easily.

  8. An instructor will provide you with a memory stick, onto which you will load your file.

  9. An instructor will then instruct on starting the printing process.
